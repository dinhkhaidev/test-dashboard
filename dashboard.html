Database & Google Calendar
Supabase setup:
1. Tạo project tại supabase.com
2. Tạo table tasks với các columns: id text, title text, desc text, date text, category text, priority text, status text, created_at bigint
3. Bật RLS và tạo policy cho phép all operations
4. Copy Project URL và anon public key vào đây
Data sẽ đồng bộ tự động lên Supabase.
Cách tạo Google credentials:
1. Vào console.cloud.google.com → tạo project
2. APIs & Services → Library → bật Google Calendar API
3. Credentials → Create Credentials → API Key → copy key
4. Create Credentials → OAuth Client ID → loại Web application
5. Tại Authorized JavaScript origins thêm origin hiện tại (ví dụ http://localhost:8080). Mở file bằng HTTP server — không chạy file:// vì OAuth không hỗ trợ.
6. Thêm tài khoản của bạn vào Test users nếu app đang ở chế độ Testing.
Database
Data ưu tiên lưu trong Supabase, tự động fallback về IndexedDB nếu chưa cấu hình. Dùng nút Export JSON trên topbar để backup.
